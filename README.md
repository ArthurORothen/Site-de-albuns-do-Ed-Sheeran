🚩 Discografia do Ed Sheeran
Um site que navega pela discografia do Ed Sheeran. O projeto altera visualmente a interface (fundo e capa) e reconstrói a lista de faixas em tempo real conforme o usuário interage com os controles.

🚀 Tecnologias Utilizadas
HTML5: Estrutura básica

CSS3: Dinâmica, incluindo manipulação de backgrounds e layouts responsivos

JavaScript:

Manipulação do DOM (createElement, appendChild).

Lógica de arrays de objetos.

Controle de fluxo com loops for e condicionais.

🧠 Desafios de Lógica Resolvidos
Durante o desenvolvimento, foquei em resolver problemas comuns de engenharia de software:

Sincronização de Dados: Utilização de um único contador para organizar a ordem de imagens, cores de fundo e arrays de músicas.

Renderização Dinâmica: Implementação de um loop que limpa a interface anterior e gera novos elementos ''li'' para que a lista de músicas corresponda ao álbum selecionado.

Navegação Circular: Lógica para que o player retorne ao primeiro álbum após o último (e vice-versa), evitando erros de índice indefinido.

🎨 Design
O projeto aplica conceitos de Teoria das Cores e UI/UX, onde cada álbum possui uma identidade visual única que se reflete em todo o plano de fundo da página.
